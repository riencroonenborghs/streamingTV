# streamingTV

An Electron app heavily inspired by Exodus.


## Dev

```
$ npm install
```

### Run

```
$ bozon start
```

### Package

```
$ bozon package
```

Builds the app for OS X, Linux, and Windows, using [electron-builder](https://github.com/electron-userland/electron-builder).
